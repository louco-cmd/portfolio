Human task management

Task dispenser for bank counselor

The Human Task Manager project was designed to improve the task distribution process for bank counselors. It aims to streamline the management of daily tasks by automatically assigning them to team members based on their availability and expertise.

By improving task assignment, this tool ensures that counselors can focus on high-priority tasks while reducing human error and inefficiencies in task allocation.

See case study

Design steps

The design process for the Human Task Manager involved several stages to refine user experience and ensure that it met the needs of bank counselors effectively.

1.

Research - workshops

We conducted workshops with bank counselors to understand their daily challenges and task management needs, ensuring that the solution would align with their workflows.

2.

Features prioritization

We prioritized features based on user feedback, focusing on automation, task tracking, and an intuitive interface for easy navigation by counselors.

3.

Sketching

Initial sketches were created to visualize the tool's layout, focusing on simple task assignment and status tracking, with clear actions for the counselors.

4.

Prototyping

We developed a high-fidelity prototype to simulate task assignment and monitor progress, making adjustments based on user feedback.

5.

User testing

User testing was conducted with bank counselors to ensure the tool was intuitive, easy to use, and helped streamline their task management process.

Research - focus group & feature prioritization

We gathered insights from bank counselors through focus groups to understand the key pain points they experienced with task management. This helped us design the solution to address their main needs effectively.
